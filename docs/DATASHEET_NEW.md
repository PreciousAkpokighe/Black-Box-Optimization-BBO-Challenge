Black-Box Optimization (BBO) Capstone Dataset
1. Motivation
This dataset was created to support the Black-Box Optimization (BBO) Capstone Project in the Imperial College Business School Professional Certificate in ML/AI programme.
The purpose of the dataset is to evaluate optimisation strategies for unknown objective functions where the mathematical relationship between inputs and outputs is hidden from the learner. The dataset supports experimentation with Bayesian Optimization, Gaussian Process Regression, Support Vector Machines, Neural Networks, Hyperparameter Tuning, CNN-inspired search strategies, and Explainable AI techniques.
2. Composition
The dataset contains:
•	8 unknown functions
•	Input vectors ranging from 2 to 8 dimensions
•	Outputs generated by the BBO platform
•	Historical observations collected across ten optimisation rounds

3. Collection Process
Data was collected through iterative interaction with the BBO challenge platform.
The optimisation process evolved through several stages:
Weeks 1–2
•	Initial exploratory sampling
•	Space-filling design
Weeks 3–4
•	SVM-assisted optimisation
•	Support vector analysis
Weeks 5–6
•	Neural Network modelling
•	SGD and Backpropagation concepts
Weeks 7–8
•	Bayesian Optimization
•	Gaussian Process Regression
Weeks 9–10
•	Explainable AI
•	Feature importance analysis
•	Transparent optimisation decisions
4. Preprocessing
Several transformations were applied:
•	Feature scaling
•	Normalisation to [0,1]
•	Log transformations for highly skewed outputs
•	Gaussian Process surrogate modelling
•	Uncertainty estimation
No personal or sensitive information exists within the dataset.
5. Intended Uses
Appropriate uses include:
•	Black-box optimisation research
•	Bayesian optimisation experiments
•	Hyperparameter tuning demonstrations
•	Surrogate model development
•	Machine learning education
Inappropriate uses include:
•	Production deployment
•	Safety-critical optimisation
•	Medical or financial decision making
